## Installation

```bash
$ npm install
```

## Running the app

```bash
#build
$ npm run build

# development
$ npm run start:dev

# one time excution
$ npm run start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
